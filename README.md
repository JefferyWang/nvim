# NvChad Custom Configs

## Usage

### Install NvChad

``` bash
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1
```

### Clone Custom Configs

``` bash
mkdir -p ~/.config/nvim/lua/custom
git clone git@github.com:JefferyWang/nvim.git ~/.config/nvim/lua/custom --depth 1
```
